## Decodelabs Internship Task 2
# 📊 Exploratory Data Analysis (EDA) in Excel – Project 2 Complete!

 This project was all about interrogating data to uncover the story before building any dashboards or models.

🔍 What I Analyzed:
An e-commerce dataset with 1,200+ orders spanning 2023–2025

📈 Key Trends I Uncovered:
* Sales peak in [MAY 2023 and JUNE 224]
* [Printers & Tablets] are the top-selling products by volume
* [Chairs] have the highest cancellation rate — worth investigating quality or delivery issues
* [Instagram / Email] brings the highest average order value — a strong channel for targeting high-spenders
* [Gift Card / Credit Card] users spend more than cash payers — indicating payment method influences basket size

📊 Tools Used:
Excel (Pivot Tables, Charts,descriptive statistics)

💡 Biggest Takeaway:
EDA isn’t just about calculating numbers — it’s about asking the right questions and letting the data guide you
